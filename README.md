# easyNotes 🚀

> AI-powered YouTube learning extension with smart summaries, timestamp-based screenshots, and visual note-taking.

easyNotes is a browser extension designed to improve the YouTube learning experience.  
Instead of switching between notes and videos, users can capture screenshots directly from YouTube, attach comments, and instantly revisit exact moments using clickable timestamps.

The extension also generates concise AI summaries from YouTube captions, helping users learn faster and organize content efficiently.

---

# 🖥️ How It Works

## 1. Install the Chrome Extension
Add the easyNotes extension to Chrome and open any YouTube video.

## 2. Capture Notes
Use the **Capture** button injected into the YouTube player to save:

- Screenshot of the current video frame  
- Exact timestamp of the video  

## 3. Add Comments
Insert custom comments between screenshots for structured learning and revision.

## 4. Generate Summary
Generate concise AI-powered summaries from YouTube captions directly from the sidebar.

## 5. Access Dashboard
Open the dashboard to create, store, organize, and browse saved notes.

# ✨ Features

- 📸 Capture screenshots directly from YouTube videos  
- ⏱️ Clickable timestamps that reopen the exact video moment  
- 📝 Add custom comments between screenshots  
- 🤖 AI-generated summaries using Groq AI + YouTube captions  
- 📂 Dashboard to create, store, and browse notes  
- ⚡ Native YouTube UI integration for seamless workflow
